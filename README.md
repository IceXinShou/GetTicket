此程式僅適用於一個賣票網站

使用方式：
1. \[網址\] 複製網站連接並填入
2. \[日期\] 若連結末包含日期，則日期欄位**可省略不填**
3. \[排序\] 看你要幾日票，通常兩日是在"1"，單日在"2"
4. \[張數\] 一次搶的票數
5. \[重複次數\] 要嘗試幾次，**若遇到張數 > 餘票會自動強制停止**
6. \[延遲\] 封包發送的間隔時間 (ms)
7. \[重複無限次\] 重複直到張數 > 餘票
8. \[無票自動調整\] 若張數 > 餘票，則調低張數嘗試清空餘票

注：若 \[重複無限次\] \[無票自動調整\] 同時開啟，則餘票為 0 將自動強制停止
